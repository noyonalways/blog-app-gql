# Blog App GraphQL

## Requirements

- User can post and publish blog content
- User can view blog content
- Authentication System
- User can see their own profile

## Technologies

- GraphQL

## Entities

- User
- Blog
- Comment
- Like

## Blog Schema

- id
- title
- content
- authorId
- createdAt
- updatedAt
- published

## User Schema

- id
- name
- email
- password
- createdAt
- updatedAt

## Profile

- id
- userId
- avatar
- bio
- createdAt
- updatedAt
